MLB Rest API

Welcome to the MLB REST API

Get All Data
www.mlbrest.com/api/all

This call will return all the mlb data contained in the site. Including every team and player along with player stats.

Get All Players By Position
www.mlbrest.com/api/:pos

This call accepts only position (p, c, if, of, dh) and will return all the active players in the league that fit these criteria.

Click here and see what you get when you ask for all pitchers.

Grab An Entire Team
www.mlbrest.com/api/team/:team

This call accepts team abbreviations. Example : bos, nyy, laa, lad, etc..

Click here and see what you get when you ask for the Red Sox.

Grab An Individual Player
www.mlbrest.com/api/team/:team/player/:name

This call accepts team abbreviations along with a player's full name to grab a player's individual stats. A SPACE IS REQUIRED BETWEEN FIRST AND LAST NAMES.

Click here and see what you get when you ask for the Red Sox pitcher Joe Kelly.
